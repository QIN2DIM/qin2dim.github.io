介绍中国大陆用户注册并使用 ChatGPT 的方法

2022-12-04
阅读时长: 5 分钟

Preview

如何在中国大陆获取 ChatGPT 的试用权限？你需要一个代理 IP 以及一个可接码的其他地区的手机号。

目前，OpenAI 未向中国大陆以及港澳台提供 ChatGPT 服务，你需要使用其他地区的代理 IP （优先使用日韩美新）进入注册页面并使用其他地区的手机号进行接码验证，注意：港澳台地区无法使用 OpenAI。

ChatGPT 屏蔽了虚拟号码，类似 Google Voice 的服务无法正常使用，但接码平台的屏蔽策略仍是基于数据库匹配，所以有些平台成了漏网之鱼，如 sms-activate。

如果你确实搞不定，可以通过 tw93/Pake 或者 反向代理 的开放接口直接上手体验（无需注册无需代理）。

快速注册

准备阶段

  1. 准备接码

    打开 sms-activate 注册一个账号，请使用真实邮箱注册。接受验证邮件可能用时略久。

  2. 注册 OpenAI 账号

    访问 OpenAI 认证界面，如果你已有账号，选择 Continue with xxx 或者点击 LogIn 即可，否则填写邮箱走流程，跟着步骤操作走就行。

    跟着操作直到 Verify Your Phone Number 界面。

  3. 购买接码账号

    使用 AliPay 对账号充值，每个接码手机号 10 卢布左右，20分钟有效，有效期内未接收到验证码，手机号将被回收，付款退回账号。

    购买 OpenAI 接码账号，选一个便宜的，这些手机号在本文首发时还剩下十几万个，用完就没了~由于账号有效期内没有活动会被回收，不用担心黄牛批量买入，这些账号用于真实注册 OpenAI。如果直接搜索服务没有看到下方的扩展栏，可以点击「显示更多国家」的拓展，SMS-ACTIVE 会时不时筛选补充接码账号，不得不说 ChatGPT 5 天时间的注册量已经超过了一百万，接码服务也算是间接受益方，他们自然不会错过这个绝佳的热点~

  4. 填写手机号，接受验证码

    账户余额足够时，点击购物车图表，自动跳转到接码界面，复制手机号填写到 ChatGPT 的验证界面中，地区选择印度，等待验证短信即可。

    ​ 选择个人用途。如果操作流程丢失，访问 ChatGPT 回到操作流程中。

首次使用

注册完成后，访问 ChatGPT (openai.com)，点击 LogIN 即可。在下方的搜索框中开始你的奇思妙想。另外，这个接口网站不太方便进入，你可以收藏这个网址，例如加入浏览器收藏夹。你也可以使用 GitHub 社区维护的 API wrapper 调用 ChatGPT。

问答案例

注意，ChatGPT 不能联网，基础知识构建工作在 2021 年完成，擅长知识型问答，它有一定的接受边界：

  1. 不能用于撰写实时性较强的时政性质的答案；
  2. 不能用于回答权限受限的问题；
  3. 不能用于回答政策、政治等专业领域的问题；
  4. 不能有效应对 2021 年之后的新知识；
  5. 禁黄赌毒；

小黄文续写

截取一段 Twitter 小黄推，让 ChatGPT 以特定作家风格续写下文。

仔细阅读 AI 给出的回复，我们不难发现，它也许并不认识余秋雨 :( 但它知道它现在是作家，需要续写一段故事。

对比前后片段，我们发现内容的广度和深度都有所拓展，但关键用词还十分 naive，同时也缺少环境氛围描写以及外貌、神态、心理的细节描写，很难充分调动读者感官，总的来说差强人意。

或者，我们可以换个角度，直接把 debuff 拉满，直接示意补充成人内容（PS：经过数个版本的迭代，2023年后的 ChatGPT 不会拦截敏感内容的输出，而是在输出敏感内容后标注并提示）。

个人感觉效果要比之前的好，故事开始有层次有梯度地推进，有起承转合，也有节奏恰到好处的落幕，总得来说已经达到了稍加润色便可投入使用的门槛标准。（吐槽：怎么回事 AI 也用拼音输入法？“的地”不分笑死）。

哲学问答

可以尝试问一些带有争议的哲学问题，如：

  • 缸中大脑

  • 电车难题

  • 造物主存在吗（死后有余生吗）

  • 基因或环境，我们的个性来自哪里

  • 人工只能是否对人类构成威胁

哲学问答的过程是不断深究的过程，你需要多伦对话引导 AI 做出更精准的回复。
